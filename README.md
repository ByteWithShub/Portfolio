
# Shubhangi Singh Tanwar | Portfolio

A creative, interactive portfolio built as a static HTML experience and deployed on Hugging Face Spaces.

[![Live Portfolio](https://img.shields.io/badge/Live%20Portfolio-Hugging%20Face-ffcc4d?style=for-the-badge&logo=huggingface&logoColor=black)](https://huggingface.co/spaces/ByteWithShub/Portfolio) 

## Overview

This portfolio is designed as more than a simple project gallery. It presents my work, process, and technical identity through an immersive interface that blends design, motion, and systems thinking.

The site highlights my work across:
- AI engineering
- machine learning systems
- computer vision
- RAG and language systems
- robotics and ROS2
- analytics and decision-focused products

At its core, the portfolio reflects how I build: from messy inputs to clear, human-ready outputs.

## Features

- Single-file static HTML portfolio
- Custom visual design system
- Animated cursor and magnetic hover interactions
- Reveal-on-scroll transitions
- Active navigation highlighting
- Subtle parallax effects
- Scroll progress behavior
- Surprise toast / hidden interaction details
- Sections for:
  - hero
  - about
  - selected work
  - process
  - credentials
  - contact

## What This Portfolio Communicates

This portfolio is built to represent my approach as an AI Engineer:
- building end-to-end systems instead of isolated models
- focusing on clarity, reliability, and usability
- connecting technical rigor with product thinking
- designing systems people can actually understand and use
